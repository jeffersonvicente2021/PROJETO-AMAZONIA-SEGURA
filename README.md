# Projeto Amazônia Segura

Sistema local de visão computacional para monitoramento de rios, comunidades e áreas estratégicas da Amazônia.

O projeto utiliza YOLOv8, OpenCV e SQLite para detectar embarcações, registrar eventos e gerar relatórios automáticos a partir de câmeras IP.

## Objetivo

Apoiar ações de segurança pública, fiscalização ambiental, defesa civil e monitoramento territorial em regiões ribeirinhas com baixa conectividade.

## Tecnologias

- Python
- YOLOv8
- OpenCV
- SQLite
- Câmeras IP/RTSP
- Relatórios automatizados
- Infraestrutura compatível com Starlink

## Funcionalidades

- Detecção de barcos, canoas e rabetas
- Registro de eventos em banco local
- Captura de imagens/snapshots
- Geração de relatórios diários, semanais e mensais
- Operação local, mesmo com internet instável
- Estrutura preparada para expansão com dashboard web
